## 机器人参数结构体

* WheelSpan：轮距（半轮轮距）
* AxleSpan：轴距（半轴距）
* GearRatio：电机减速比
* EncoderAccuracy：编码器精度（编码器线数）

```c
typedef struct  
{
    float WheelSpan;
    float AxleSpan;
    int GearRatio;
    int EncoderAccuracy;
}Robot_Parament_Init;
```
