## 简述

YOLO Nano是一种高度紧凑的深度卷积神经网络，用于目标检测任务。它是基于YOLO系列单镜头目标检测网络架构的设计原则的原则网络设计原型，并结合了机器驱动的设计探索，创造了一个紧凑的网络，具有高度定制的模块级宏观架构和为嵌入式目标检测任务量身定制的微架构设计。该模型大小约为4.0MB，比Tiny YOLOv2和Tiny YOLOv3小15.1倍和8.3倍以上。
